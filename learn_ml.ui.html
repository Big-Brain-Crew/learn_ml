class learn_ml.ui.App.LearnML(parent=None)[source]¶

Bases: PySide2.QtWidgets.QMainWindow

Defines UI element of the Learn ML project.

This class extends the QMainWindow class and serves as the entry point for the application. It will contain all of the main UI elements used to interact with our application.

deploy()[source]¶

Deploys the loaded model to the coral board.

Currently, deploy can only deploy the compiled model and does not handle converting the model to a tflite edgetpu model. This functionality will be added once USB serial communication with the coral board is achieved.

load()[source]¶

Opens the file dialog window to allow the user to load a project.

This is currently configured to select a project directory. There are no checks performed to ensure that the project directory is valid.

load_from_file(project_dir)[source]¶

Contains operations to execute when a project is loaded

Currently, the only operation performed is updating the current_file_name property.

populateMenu()[source]¶

Adds all of the options to the menu bar.

Currently, the only available options are the File option, which contains options to create a new project or open an existing project.

set_title()[source]¶

Sets the title of the application.

The title will be Learn ML - {Current Project Name}.If the project has been modified, there will be an asterisk following the project name.

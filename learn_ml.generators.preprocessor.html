learn_ml.generators.preprocessor.pipeline_generator module¶

This script generates a dataset pipeline script based off a JSON config file. This config file is generated from user inputs on the frontend. The generated script describes a DatasetPipeline object that can then be used for agnostic model training.

add_operation(op_name, args={})[source]¶

Add a preprocessing operation.

A list of all allowable operations can be found as methods for the tf.data.Dataset class

(https://www.tensorflow.org/api_docs/python/tf/data/Dataset)

Parameters

  • op_name – Name of the operation. Equivalent to a tf.Dataset method name.

  • args – Dictionary of arguments (param : value) passed to the method. The value doesn’t always correspond to the actual argument so that functionality can be abstracted from specific machine learning libraries. Check variable_map.json for all values and their representations.

  • usage (Example) –

  • = { (args) – “map_func” : “normalize_img”, “num_parallel_calls” : “autotune” }
